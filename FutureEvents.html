August 12 - Chess 2.0 Release Date

The sequel to Chess is coming out after 1,500 years of the same game. After the same game not having the same update for 1 and a half millennia the creators of chess have officially announced the new game incoming. Experts have predicted the following new changes to be added.

  1. New rocket launcher piece, this piece will be able to attack pawns
  2. New 3-D feature where the board will rotate like a rubik's cube after every few moves
  3. New spy glasses that will alllow for the queens to beam through two pieces
  4. After capture, each piece drops aura which can be used to buy powerups
  5. Removing the knights from the game
  6. Quality of life changes
